# Hardware Machine

## Requirements

A 5-year expected lifetime machine:
- CPU/GPU: as fast as possible & affordable
- Memory: 64G if affordable, 32G minimal
- Hard drive: 1TB system, 1TB data, 3TB NAS/Cloud
- Sound:
- Camera:
- Mic: Blue Yeti
- Mouse & keyboard
- Monitors: 2 42" monitors
- Digital pen: GAOMON S620
